# National Charter School Statistics dashboard concept
National charter school statistics dashboard concept demo.

# What it does
Mapping, donut and bar charts for 6 levels of fine-grain and drill-down: national, state, school district, congressional district, management company, and school.

Can slide a slider to change year of data. Can export a snapshot page optimized for browser PDF printing.

Completely client side technology, with no framework ( just jQuery and JavaSript #_# ).
